ValenFind – TryHackMe Write-Up
Introduction
The ValenFind room in TryHackMe is a beginner-friendly web challenge that focuses on basic enumeration and reconnaissance skills. The main objective of this room is to locate a hidden flag within a Valentine-themed website by exploring beyond what is immediately visible.
This challenge helped me practice important web penetration testing techniques such as inspecting source code and performing directory enumeration.

Step 1: Deploying the Machine
I started by deploying the machine on TryHackMe. After the machine was fully loaded, I copied the target IP address provided by the platform. This IP address is used to access the vulnerable web server.

Step 2: Initial Website Inspection
I entered the target IP address into my browser to access the website. The webpage had a simple Valentine theme and did not show any obvious sensitive information or flags on the homepage. At first glance, everything looked normal.
Since no flag was visible, I knew I needed to investigate deeper.

Step 3: Checking the Page Source
My next step was to inspect the HTML source code of the webpage. I right-clicked on the page and selected “View Page Source.” I carefully examined the code for hidden comments, suspicious links, or any clues left by the developer.
Although the source code provided some structure of the page, the flag was not directly visible. This indicated that the flag might be hidden in another directory or page.

Step 4: Directory Enumeration
Because the flag was not on the homepage, I performed directory brute-forcing using a tool like gobuster. This tool helps discover hidden folders or files that are not linked publicly on the website.
By scanning the target IP with a common wordlist, I was able to find a hidden directory. This directory was not accessible from the main page, which means it required enumeration to discover it.

Step 5: Finding the Flag
After discovering the hidden directory, I accessed it manually through the browser. Inside that page, I found the flag. In some cases, the flag may be directly displayed on the webpage, or it may be hidden within the page source code.
Once I located the flag, I copied it and submitted it on TryHackMe to complete the challenge.

Conclusion
The ValenFind room is a great beginner challenge because it teaches the importance of proper enumeration and investigation. Even if a website looks simple and harmless at first, important information may be hidden in directories or within the source code. This room strengthened my understanding that in cybersecurity, looking deeper and not relying only on what is visible is very important.
